# E-Commerce Platform Search Function

## Aim

To implement Linear Search and Binary Search algorithms in Java.

## Files

- Product.java
- SearchAlgorithms.java
- Main.java

## Description

This program demonstrates searching for a product using Linear Search and Binary Search. The products are stored in an array, and the array is sorted before performing Binary Search.

## Time Complexity

| Algorithm | Best Case | Average Case | Worst Case |
|-----------|-----------|--------------|------------|
| Linear Search | O(1) | O(n) | O(n) |
| Binary Search | O(1) | O(log n) | O(log n) |

## Output

```
Linear Search:
ID: 102, Name: Mouse, Category: Electronics

Binary Search:
ID: 102, Name: Mouse, Category: Electronics
```
