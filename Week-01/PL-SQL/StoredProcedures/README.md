# Stored Procedures

## Aim

To implement basic stored procedures in MySQL for different banking operations.

## Scenarios Covered

### 1. ProcessMonthlyInterest
- Applies 1% interest to all customer account balances.

### 2. UpdateEmployeeBonus
- Updates employee salaries by adding a bonus percentage for a selected department.

### 3. TransferFunds
- Transfers money from one account to another after checking whether the sender has sufficient balance.

## Tables Used

- Accounts
- Employees

## Concepts Used

- Stored Procedures
- Parameters
- IF Condition
- UPDATE Statement
- Transactions
- COMMIT
- ROLLBACK
