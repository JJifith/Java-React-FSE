# Control Structures

## Objective

To practice PL/SQL control structures using MySQL.

## Scenarios

1. Apply a 1% loan interest discount for customers above 60 years.
2. Promote customers with a balance greater than 10000 to VIP.
3. Display reminders for customers whose loan is due within 30 days.

## Concepts Used

- Stored Procedures
- Variables
- Cursor
- LOOP
- IF
- UPDATE
