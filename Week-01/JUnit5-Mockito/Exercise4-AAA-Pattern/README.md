# Exercise 4 - Arrange-Act-Assert (AAA) Pattern

## Aim

To understand the Arrange-Act-Assert (AAA) testing pattern and use setup and teardown methods in JUnit.

## Steps

- Created a Calculator class.
- Wrote a unit test using the AAA pattern.
- Used `@BeforeEach` to initialize the test object.
- Used `@AfterEach` to clean up after each test.

## Files

- Calculator.java
- CalculatorTest.java
- pom.xml
- README.md

## Result

The test executed successfully using the AAA pattern along with setup and teardown methods.
