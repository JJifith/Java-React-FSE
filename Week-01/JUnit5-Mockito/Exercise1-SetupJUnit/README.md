# Exercise 1 - Setting Up JUnit

## Aim

To configure a Java Maven project with JUnit 5 for writing and executing unit tests.

## Steps

- Created a Maven Java project.
- Added JUnit 5 dependency in `pom.xml`.
- Configured Maven Surefire Plugin.
- Created the test source folder.
- Created a sample test class.

## Files

- pom.xml
- src/main/java
- src/test/java
- README.md

## Result

JUnit 5 was successfully configured and the project is ready for writing unit tests.
