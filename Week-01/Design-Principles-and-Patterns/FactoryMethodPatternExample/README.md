# Factory Method Pattern Example

## Aim

To implement the Factory Method Design Pattern in Java.

## Files

- Document.java
- WordDocument.java
- PdfDocument.java
- ExcelDocument.java
- DocumentFactory.java
- WordFactory.java
- PdfFactory.java
- ExcelFactory.java
- Main.java

## Description

This program demonstrates the Factory Method Pattern by creating different types of documents such as Word, PDF, and Excel using their respective factory classes.

## Output

```
Opening Word Document...
Opening PDF Document...
Opening Excel Document...
```
