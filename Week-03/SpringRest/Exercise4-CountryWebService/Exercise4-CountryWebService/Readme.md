# Exercise 4 - REST Country Web Service

## Aim

To create a REST API that returns India country details loaded from a Spring XML configuration file.

## Technologies Used

- Java
- Spring Boot
- Spring Web
- Spring Core
- Maven

## Endpoint

GET /country

Example:

http://localhost:8083/country

## Response

```json
{
  "code": "IN",
  "name": "India"
}
```
