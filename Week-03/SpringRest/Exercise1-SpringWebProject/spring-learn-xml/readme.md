# Exercise 2 - Spring Core: Load Country from Spring Configuration XML

## Aim
To configure a Country bean using Spring XML configuration and load it using the Spring IoC container.

## Technologies Used
- Java 17
- Spring Boot
- Spring Core
- Maven

## Files
- Country.java
- country.xml
- SpringLearnApplication.java
- application.properties

## Output
The application loads the Country bean from `country.xml` and prints its details.

Example Output:

Country : Country{code='IN', name='India'}
