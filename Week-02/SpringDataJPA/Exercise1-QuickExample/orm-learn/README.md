# Exercise 1 - Spring Data JPA Quick Example

## Aim

To create a basic Spring Data JPA application that connects to MySQL and retrieves data using JpaRepository.

## Technologies Used

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## Project Structure

- Country Entity
- CountryRepository
- CountryService
- OrmLearnApplication

## Output

- Connected to MySQL database.
- Created `country` table using Hibernate.
- Retrieved country records using JPA Repository.
